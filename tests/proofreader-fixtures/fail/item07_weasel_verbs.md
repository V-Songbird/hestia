# Analyze Code

To get started, you might want to use `Read` to load the relevant source files. When searching for usage patterns, it would be helpful to consider using `Grep`. You can also use `Glob` if you want to discover files by pattern.

Once the files are loaded, review the code for common issues such as inconsistent naming, missing error handling, and dead code. Generate a summary report for the user.
